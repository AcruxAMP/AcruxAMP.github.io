アカペラ練習用MIDIプレーヤー Acrux AMP の更新履歴

v0.1.0-beta.7β

2026-07-14

修正

  • 再生が不安定（音の途切れ・周波数のゆらぎ・BPMのドリフト・バックグラウンドで停止）だった問題を修正。実音声の出力経路をスピーカー直結に戻した
  • メトロノームの強調を1拍目のみに変更

変更

  • メディア通知・バックグラウンド維持を、実音声を通さない無音要素方式に変更（Android の通知コントロールは維持しつつ、音質に影響しない）
  • モバイルでの音切れ対策として同時発音数の上限を設定、波形の倍音数を調整

v0.1.0-beta.6β

2026-07-14

変更

  • シンセの音色を改善: アタックを強化し、倍音を加えた波形に変更。三角波よりも明るく、音程の輪郭が聞き取りやすい

修正

  • メトロノームが 6/8 などの分母8の拍子でも4分音符で刻まれていた問題を修正。6/8 は8分×6（1拍目=強・4拍目=中の強弱2段）で刻み、曲中の拍子変化（変拍子）にも追従する
  • Android で通知・ロック画面の再生コントロールが表示されなかった問題を修正（iOS の画面ロック中の再生継続にも効く経路になる）

v0.1.0-beta.5β

2026-07-10

機能

  • 楽譜の表示サイズ変更（50〜150%）。楽譜右下の −/＋ で調整、％表示のタップで100%に戻る。縮小するとより多くの小節を表示できる。倍率は保存される
  • Media Session 対応: ロック画面・通知領域の再生/一時停止/停止コントロールから操作可能に

変更

  • 小節幅を音価ベースで算出（4分 > 8分 > 16分と広さが変わり、リズムが見た目で読める）
  • OSのピンチズームを無効化（楽譜の拡縮はアプリ内のズームコントロールで行う）
  • パッチノートをPWA内の画面遷移ではなく外部ブラウザで開くように変更

修正

  • バックグラウンド再生を安定化（出力バッファ・先読みの拡大、iOSの割り込みでサスペンドされた音声の画面復帰時の自動再開、背面中も曲末で自動停止）
  • メイントラックの強調マーカーが横スクロール中にずれる問題を修正
  • 再生中の小節がスティッキーガターに隠れる問題を修正（自動スクロール位置を補正）
  • 符幹が下向きの音でスタッカート点が符頭側に付かない問題を修正

v0.1.0-beta.4β

2026-07-10

機能

  • 音部記号・調号・拍子記号を楽譜の左端にスティッキー表示（横スクロールしても流れず固定）。転調・変拍子を過ぎると、その時点で有効な記号に差し替わる
  • 曲中の転調に対応。転調した小節に新しい調号を表示（前の調号を打ち消すナチュラル付き）。音名表記・臨時記号の判定も小節ごとの調号に追従

修正

  • 複数の調号イベントを持つMIDIで先頭の調号しか反映されなかった問題を修正
  • メイントラックの強調マーカーが横スクロール時に描画位置からずれる場合があった問題を修正

v0.1.0-beta.3β

2026-07-09

機能

  • 曲中のテンポ変化に対応（正しい小節で加速・減速。BPM変更は全体を倍率でスケール）
  • 変拍子（曲中の拍子変化）に対応。拍子変化した小節に拍子記号を表示（7/8などの非整数拍も対応）
  • スタッカートの表示に対応
  • 拍子記号の表示（曲頭・拍子変化時）
  • メイントラックの左端に強調用の縦棒マーカーを常時表示（横スクロールに追従）

変更

  • 位置追跡・小節ハイライト・シークをティックベースに統一（テンポ・拍子変化に非依存）
  • スタッカート点を符幹の反対側（符頭側）に配置

v0.1.0-beta.2β

2026-07-09

機能

  • ダークテーマ・ライトテーマに対応（デフォルトはシステム設定に追従、Settingsで切り替え）
  • 音部記号に「自動（パートの音域から選択）」を追加し、デフォルトに設定。複数パート表示ではパートごとに最適な記号（ト音／ト音8vb／ヘ音）で表示
  • BPM設定に「元のBPMへリセット」ボタンを追加
  • 曲の最後まで再生したら自動で停止
  • パッチノートページ（このページ）を追加

変更

  • 複数パートの楽譜表示をトラック順に変更（メインパートを先頭に固定しない）

修正

  • BPMの変更が楽譜ごとの設定に保存されない問題を修正

v0.1.0-beta.1β

2026-07-08 — 初回β版

機能

  • MIDIファイルの読み込み・保存・一覧管理（改名・削除）
  • 再生・一時停止・シーク、BPM変更（再生中も即時反映）
  • パート別ミュート、メインパート強調（メイン／サブ音量の個別調整）
  • パーカッショントラックの再生
  • メトロノーム（1拍目アクセント付き・音量調節）
  • 楽譜上の小節タップ／テンキーによる小節ジャンプ
  • 楽譜表示: 横スクロールレイアウト・複数パート同時表示・再生位置ハイライトと自動スクロール・音部記号3種
  • 歌詞表示（UTF-8 / Shift-JIS 自動判別、カラオケ形式対応、重複解消）
  • 楽譜ごとの設定保存（BPM・ミュート・メインパート・表示パート）
  • PWA対応（オフライン動作）

楽譜エンジン

  • 拍グリッド量子化によるリズム解析（16分／32分／8分3連／16分3連を自動判別、タイミングの揺れを補正）
  • 連符（3連符）・タイ・付点音符（最小32分）
  • 調号（♯系／♭系）と臨時記号（♯・♭・♮、小節内持続ルール対応）

修正

  • 日本語トラック名・タイトルの文字化けを修正
  • メトロノームOFF中に音量を変更するとクリックが鳴り出す問題を修正
  • 休符を挟んだ音符がビームで連結される問題を修正
  • ヘ音記号表示時に休符が五線から浮く問題を修正

既知の制約

  • 5連符・7連符などの特殊連符は未対応
  • 転調・拍子変更の表示は先頭のもののみ
  • 強弱記号は表示されない（MIDIに記号情報が存在しないため）
